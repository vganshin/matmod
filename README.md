# How to run strategies

> Warn: for chicken game only.

```
node src/index.js gameName [release]
```

Allowed game names: `prisoner`, `chicken`, `goods`.

# How to run visualizer

Run command
```
node src/server.js
```

and go to http://localhost:8080.

# How to run shower

> Warn: for chicken game only.

```
node src/shower.js
```
